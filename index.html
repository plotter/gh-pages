The Plotter Project

Modular set of open source tools and supporting community for building future-proof, lightweight, extensible spatial information systems that can be deployed to desktop apps (on windows, mac, and linux), websites, and mobile apps (on iOS, Android, Windows Phone, and other mobile platforms)

id-tools

tools for providing authentication and role-based authorization within applications

schema-tools

tools for managing application data schema across and within data sources

pak-tools

tools for managing extensible *paks* of functionality within an application

data-tools

tools for managing data sources

file-tools

tools for managing (virtualized) public and private folders and files

map-tools

tools for building and extending spatial visualization and analysis with a plugin model for incorporating Leaflet, ESRI, Google, Bing and other GIS technologies

search-tools

tools for creating, editing, and running structured and unstructured (and optionally spatial) data searches

chart-tools

tools for creating interactive, animated data visuals supporting a wide variety of chart types

video-tools

tools for authoring, managing, and viewing remixed web video, audio and images in *mashups*. drag and drop content from the web and add comments and links

three-d-tools

tools for authoring, editing, managing, and displaying webgl 3d content
